# 📚 Library Management System (Read & Rise)

A modern, responsive Digital Library Management System built using **C#** and **.NET MVC**. The application offers a tailored and seamless experience for both library administrators and regular readers, ensuring efficient content management and an engaging book discovery journey.

---

## 🚀 Key Features

### 👨‍💼 Administrator Panel (Content Management)
The Admin has full control over the library's catalog through a comprehensive dashboard:
* **Add New Books:** Easily expand the library by adding new titles with full details (Author, Category, Pages, Language, Description, and PDF files).
* **Edit & Update:** Modify any existing book's details, description, or attributes at any time.
* **Delete Books:** Remove outdated or unavailable books safely from the database.
* **Advanced Content Management:** Complete control over managing descriptions, view-more actions, and backtracking.
* **Search & Filter:** Quickly find and manage any book using targeted search and category filtering.

### 👤 User / Reader Experience
Regular readers enjoy a clean, distraction-free environment to explore and read:
* **Secure Authentication:** Easy Sign-In and Sign-Up options to create a personalized reader account.
* **Read Descriptions:** Access full metadata and detailed descriptions of any book before downloading.
* **Advanced Search:** Instantly look up specific books by title, author, or keywords.
* **Smart Filtering:** Filter books by specific genres or tags (e.g., *Policy*, *Romance/Love*, *Fiction*, etc.) to match their reading mood.
* **Direct PDF Downloads:** One-click download feature to save the book directly to their device as a PDF.

---

## 🛠️ Technologies Used

* **Backend Language:** C#
* **Framework:** .NET MVC
* **Database:** SQL Server (Entity Framework / ADO.NET)
* **Frontend:** HTML5, CSS3, Bootstrap, JavaScript (Responsive layout for all devices)
